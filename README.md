# IS3020 Python Programming — Weekly Assignments

## How to Use This Repo

### First Time Setup
1. **Fork** this repo to your own GitHub account
2. Clone your fork: `git clone https://github.com/YOUR-USERNAME/python-assignments.git`
3. Open the folder in PyCharm

### Each Week
1. Pull any updates: `git pull`
2. Go to the week's folder (e.g., `week01/`)
3. Read `instructions.md`
4. Complete `solution.py`
5. Test your code locally by running it
6. Commit and push:
   ```bash
   git add -A
   git commit -m "Complete week 1"
   git push
   ```

### Deadlines
Each week's assignment is due by **Sunday 11:59 PM**. Your last commit before the deadline counts.
