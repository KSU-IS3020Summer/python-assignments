# Week 10: Final Project

## Overview
Build a **Student Management System** that combines everything you've learned.

## Requirements
Your program must include:

### 1. Classes (Week 9)
- A `Student` class with name, ID, and grades
- A `Course` class that manages students

### 2. File I/O (Week 8)
- Save student data to a file
- Load student data from a file

### 3. Functions (Week 7)
- Menu-driven interface using functions
- Input validation

### 4. Data Structures (Weeks 4-6)
- Use lists, dictionaries, or both to store data

### 5. Control Flow (Weeks 2, 5)
- Loops for the menu system
- Conditionals for validation and logic

## Features to Implement
Your system should support:
1. Add a new student
2. Add grades to a student
3. View all students and their averages
4. Find the top student
5. Save data to file
6. Load data from file
7. Exit

## Grading Criteria
- Code runs without errors (30%)
- All 7 features work (40%)
- Clean code: functions, good variable names, comments (15%)
- Uses a class properly (15%)

## Submission
Complete `solution.py` with your full program.
